# Wedding Invitation Page — Content Spec

---

The page should be in Russian.

## Display (inform guests)

### Required
- Couple's names: Евгений и Анастасия
- Date and time — ceremony: 8 августа 2026
- Venue addresses with map links: станица Коневская, Краснодарский край
- Dress code: заполни плейсхолдером
- RSVP deadline

### Optional
- Day schedule (approximate)
- Parking / transport info
- Nearby hotels for out-of-town guests
- Gift preferences (if any)
- Contact for questions (organizer, not the couple)
- Photo of the couple

---

## RSVP Form (collect from guests)

### Required
- Full name
- Attendance confirmation (yes / no)
- Number of guests (if +1 is allowed)

### Useful for logistics
- Dietary restrictions / allergies
- Menu choice (if applicable)
- Transfer needed (yes / no)
- Hotel accommodation needed (yes / no)

### Optional
- Song request
- Message to the couple

---

## Technical Notes

- Mobile-first layout
- Form responses auto-saved to a spreadsheet (e.g. Google Forms → Sheets)
- RSVP deadline: 2–3 weeks before the event

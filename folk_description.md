# ТЗ: свадебное приглашение с кубанской казачьей тематикой — два варианта

> Документ для Claude Code. Оба варианта — HTML/CSS/JS, один файл, никаких зависимостей.
> Контент-плейсхолдеры отмечены `[...]`.

---

## Общее для обоих вариантов

**Назначение:** Лендинг-приглашение на свадьбу в казачьем стиле. Гости открывают ссылку и получают всю информацию: имена, дата, место, программа, RSVP.

**Структура страницы (секции):**
1. **Hero** — имена жениха и невесты, дата свадьбы
2. **История** — короткий рассказ о паре («как познакомились»)
3. **День свадьбы** — программа: три блока (Церемония / Банкет / Гуляния)
4. **Место** — адрес, карта-заглушка или схема проезда
5. **RSVP** — форма подтверждения (имя + присутствие + кнопка «Подтвердить»)

**Технические требования:**
- Чистый HTML5 + CSS (custom properties) + vanilla JS
- Один файл `index.html`, без внешних зависимостей кроме Google Fonts
- Responsive: корректно от 375 px до 1440 px
- Плавный скролл, hover-состояния, одна CSS-анимация при загрузке
- Шрифты подключаются через `@import` из Google Fonts
- RSVP-форма: при submit показывает inline-сообщение «Спасибо, ждём вас!» (без реального бэкенда)

# Вариант A — «Лайн-арт»

**Концепция:** Светлое, воздушное приглашение, где казачья традиция присутствует как *орнамент*, *контур*, *деталь* — не как иллюстрация. Тепло и нежность через пшеничную палитру, изящная антиква. Казачий мотив — сдержанный акцент, не доминанта.

**Ключевое запоминаемое решение:** Имена жениха и невесты разделены тонким SVG-мотивом — геометризированным цветком подсолнуха (символ Кубани), нарисованным штрихами. Он «прорисовывается» при загрузке через `stroke-dashoffset`.

---

## A. Палитра

```css
--bg:           #F7F2E8;  /* пшеничный, тёплый офф-вайт */
--bg-alt:       #EDE8DC;  /* чуть темнее, для чередования секций */
--ink:          #1A1410;  /* почти чёрный, тёплый */
--accent-red:   #C0392B;  /* казачий красный — только для лампаса и акцентов */
--accent-gold:  #B8860B;  /* тёмное золото, орнамент, SVG */
--rule:         #D4C9B0;  /* линейки, разделители */
--muted:        #7A7060;  /* вспомогательный текст */
```

---

## A. Типографика

```
Имена / главные заголовки:  «Cormorant Garamond», weight 300, italic, tracking +0.02em
                              — антиква с острыми засечками, перекликается с вязью
H2 секций:                   «Cormorant Garamond», weight 400, нормальный
Основной текст:               «Jost», weight 300–400, 16–18 px, line-height 1.75
Время / место / лейблы:       «Jost», uppercase, letter-spacing 0.15em, weight 500, 11–12 px
```

---

## A. Hero-секция

**Макет:** Центрированный, 100vh. Текст занимает центральную колонку max-width 640 px.

**Вертикальная полоса-лампас:**
```css
.лампас {
  position: fixed;
  left: 24px;
  top: 0;
  width: 3px;
  height: 100vh;
  background: var(--accent-red);
  transform-origin: top;
}
```
При скролле — `scaleY` через JS `window.scrollY`, эффект «растущей полосы» прогресса.

**Текст и порядок элементов:**
```
[дата свадьбы]                    ← Jost uppercase, 12 px, letter-spacing 0.25em, --muted
                                     пример: «22 АВГУСТА 2025»

[Имя невесты]                     ← Cormorant Garamond italic, 72 px desktop / 40 px mobile
                                     --ink

    [Аккуратный SVG-цветок подсолнуха (не нарочито)]
                                  ← ~60×60 px, stroke-only, --accent-gold
                                     анимация: stroke-dashoffset, 1.2 s при загрузке

[Имя жениха]                      ← то же, что имя невесты

[Приглашаем вас разделить        ← Jost, 18 px, --muted, max-width 420 px
 с нами этот день]                  под именами

[↓ Подробнее]                     ← Jost uppercase, 11 px, --muted; плавный скролл к секции 2
```

Горизонтальный SVG-бордюр внизу Hero: повторяющийся мотив «ромб + крест», 12 px высота, `--rule`.

---

## A. Секция «День свадьбы»

**Макет:** `display: grid; grid-template-columns: repeat(3, 1fr); gap: 2px;`

Карточки разделены только gap. Фон `--bg-alt`. При hover — `background: --ink`, текст инвертируется. Переход 200 ms.

```
Карточка (×3):
  [Время]                 ← Jost uppercase, 11 px, --muted, letter-spacing 0.15em  («14:00»)
  [Название события]      ← Cormorant, 26 px, --ink                                («Церемония»)
  [Место / описание]      ← Jost, 14 px, line-height 1.6
```

---

## A. Секция «Место»

**Макет:** Две колонки 40/60. Левая — текстовый блок с адресом. Правая — SVG-схема: стилизованная карта-заглушка (прямоугольник с крестом дорог и пином), `stroke --accent-gold`.

```
[Название площадки]       ← Cormorant italic, 28 px
[Адрес]                   ← Jost, 16 px, --muted
[Ссылка на Яндекс.Карты]  ← Jost uppercase, 11 px, --accent-red, underline при hover
```

---

## A. Секция RSVP

Фон `--bg`. Над секцией — горизонтальный SVG-бордюр (тот же, что внизу Hero).

Содержимое описано в forms_description.md

---

## A. Анимации

| Элемент | Поведение |
|---|---|
| SVG-подсолнух в Hero | stroke-dashoffset, 1.2 s, ease-out, при загрузке |
| Имена | fade-in + translateY(16px)→0, staggered 200 ms |
| Карточки «День» | fade-in при scroll (IntersectionObserver), staggered 100 ms |
| Лампас | scaleY через JS scroll listener, origin: top |
| Форма RSVP | при submit: форма fade-out, сообщение fade-in |

# Вариант B — «Казачий пир»

**Концепция:** Тёмный, праздничный, торжественный. Свадьба как *гуляние*, *пир*, *обряд*. Золото на тёмном, как свечи на вечернем столе. Казачья эстетика через орнамент черкески и атмосферную Canvas-текстуру. Вдохновение: винная этикетка Studio13 «Вина Кубани», MORTESHKA website.

**Ключевое запоминаемое решение:** Hero занимает 100vh. Имена жениха и невесты разделены казачьей звездой — SVG с двойным контуром, нарисованной золотом. Фон — Canvas-текстура: тёмная основа, золотой шум, тихий шеврон. Fluid-типографика: имена во всю ширину через `clamp()`.

---

## B. Палитра

```css
--bg:           #0E0C09;  /* почти чёрный, тёплый */
--surface:      #1A1610;  /* фон карточек и секций */
--border:       #2E2820;  /* тонкие разделители */
--text:         #E8E0D0;  /* тёплый off-white */
--muted:        #7A7060;  /* вспомогательный */
--gold:         #C9A84C;  /* основной акцент */
--gold-dim:     #8A6E30;  /* орнамент, фоновые детали */
--red:          #8B1A1A;  /* глубокий красный, только акценты */
```

---

## B. Типографика

```
Имена / H1:       «Playfair Display», weight 400, italic
                   — сильная контрастная антиква, работает на тёмном фоне
H2 секций:         «Playfair Display», weight 400, нормальный
Текст:             «Raleway», weight 300, 16 px, line-height 1.75
Лейблы / время:    «Raleway», uppercase, weight 600, letter-spacing 0.2em, 11–12 px
```

---

## B. Hero-секция

**Фон:** `<canvas id="hero-bg">`, 100% × 100%, JS:
- Основа: `#0E0C09`
- Золотой шум: 3 000 случайных точек `rgba(200,160,60,0.03–0.07)`
- Шеврон-паттерн: мотив `V`, шаг 60 px, `strokeStyle: rgba(201,168,76,0.07)`, 1 px — едва видимый

**Макет:** Центрирован горизонтально и вертикально.

```
[«22 АВГУСТА 2025»]           ← Raleway, 12 px, uppercase, letter-spacing 0.3em, --gold
                                 eyebrow над именами

[Имя невесты]                  ← Playfair Display italic, clamp(36px, 7vw, 96px), --text
                                 ~70–80% ширины контейнера

    [SVG казачья звезда]       ← ~48×48 px, двойной контур, stroke --gold
                                 тихое вращение: @keyframes spin, 20 s, linear, infinite

[Имя жениха]                   ← то же, что имя невесты

[«Приглашаем вас»]             ← Raleway, 16 px, --muted, max-width 400 px
                                 под именами, центр

[ПОДРОБНЕЕ ↓]                  ← border 1px --gold, padding 12px 36px
                                 hover: background --gold, текст --bg, 300 ms
```

Снизу Hero — линия `--gold-dim` + пульсирующая стрелка «↓» в `--gold`.

---

## B. Секция «День свадьбы»

**Сетка:** `grid-template-columns: repeat(3, 1fr); gap: 1px; background: --border;`
Тонкие «золотые» линии между карточками.

Каждая карточка: фон `--surface`. Сверху — «шпора»: горизонтальная золотая черта 40 px. При hover — `translateY(-4px)` + shadow, «шпора» растягивается до 100% ширины.

```
[Иконка SVG 32×32]        ← линейная, stroke --gold
[Время]                   ← Raleway uppercase, 11 px, --gold, letter-spacing 0.2em
[Название]                ← Playfair Display, 22 px, --text   («Церемония»)
[Место / описание]        ← Raleway, 13 px, --muted
```

SVG-иконки (инлайн, stroke-only):
- Карточка «Церемония»: кольца — два пересекающихся круга
- Карточка «Банкет»: бокал
- Карточка «Гуляния»: нагайка / витая линия

---

## B. Секция «Место»

**Макет:** Полная ширина, фон `--bg`. Заголовок центрирован. Под ним — две колонки.

Левая:
```
[Название площадки]       ← Playfair Display italic, 28 px, --text
[Адрес]                   ← Raleway, 15 px, --muted
[Ссылка на карты]         ← Raleway uppercase, 11 px, --gold; underline при hover
```

Правая — SVG-схема-заглушка: тёмный прямоугольник с пересечением дорог и пином, `stroke --gold-dim`.

---

## B. Секция RSVP

Фон `--surface`. Заголовок H2 центрирован. Под заголовком — горизонтальный орнаментальный разделитель: SVG из 5 ромбов, `--gold-dim`.

Содержимое описано в forms_description.md

---

## B. Анимации

| Элемент | Поведение |
|---|---|
| Canvas Hero | рисуется при загрузке, 600 ms |
| Имена | clip-path reveal: `inset(0 100% 0 0)` → `inset(0 0% 0 0)`, staggered 300 ms |
| SVG-звезда | `@keyframes spin`, 20 s, linear, infinite |
| «Шпора» на карточках | width 40px → 100% при hover, 200 ms ease |
| Пульс стрелки Hero | opacity 1→0.3→1, 2 s infinite |
| Форма RSVP | при submit: fade-out → fade-in сообщения |

## Сравнительная таблица

| | Вариант A «Лайн-арт» | Вариант B «Казачий пир» |
|---|---|---|
| **Фон** | Пшеничный тёплый `#F7F2E8` | Почти чёрный `#0E0C09` |
| **Тон** | Нежный, воздушный, дневной | Торжественный, вечерний, пиршественный |
| **Шрифты** | Cormorant Garamond + Jost | Playfair Display + Raleway |
| **Казачий элемент** | Лампас + SVG-подсолнух | Canvas-шеврон + вращающаяся звезда |
| **Символ между именами** | Геометрический подсолнух | Казачья звезда |
| **Карточки «День»** | Инверсия при hover | Золотая «шпора» + приподнимание |
| **Анимация Hero** | stroke-dashoffset | Canvas render + clip-path reveal |
| **Настроение** | Утренняя роса, поле, лёгкость | Свечи, золото, ночное гуляние |
| **Сложность кода** | Средняя | Средняя–высокая (Canvas) |

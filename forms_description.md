## RSVP Form (collect from guests)

### Required
- Full name
- Attendance confirmation (yes / no)
- Number of guests (if +1 is allowed)

### Useful for logistics
- Dietary restrictions / allergies
- Menu choice (if applicable)
- Transfer needed (yes / no)
- Hotel accommodation needed (yes / no)

### Optional
- Song request
- Message to the couple

---

## Technical Notes

- Mobile-first layout
- Form responses auto-saved to a spreadsheet (e.g. Google Forms → Sheets)
- RSVP deadline: 2–3 weeks before the event
